# `isOwnerConnected`

Indicates whether a signer who is owner of the current Safe is connected. This flag is useful for checking whether the connected signer is authorised to perform any signing method with the Safe, such as sending transactions or signing messages.

## Usage

{/* <!-- vale off --> */}

<Tabs items={['example.tsx']}>
  <Tabs.Tab>
    ```typescript
    import { useSafe } from '@safe-global/safe-core-sdk-react-hooks'

    const IsOwnerConnected = () => {
      const { isOwnerConnected } = useSafe()

      return (
        <>
          <h3>{isOwnerConnected ? 'Owner is connected' : 'Owner is not connected'}</h3>
        </>
      )
    }

    export default IsOwnerConnected
    ```
  </Tabs.Tab>
</Tabs>
