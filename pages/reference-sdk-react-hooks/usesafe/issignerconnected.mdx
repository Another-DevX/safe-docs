# `isSignerConnected`

Indicates whether a signer is connected. Check this before using any method that requires a signer, such as sending transactions or signing messages.

**Note**: This flag does not indicate whether the connected signer is an owner of the Safe. Use [`isOwnerConnected`](./isownerconnected.mdx) for that purpose.

## Usage

{/* <!-- vale off --> */}

<Tabs items={['example.tsx']}>
  <Tabs.Tab>
    ```typescript
    import { useSafe } from '@safe-global/safe-core-sdk-react-hooks'

    const IsSignerConnected = () => {
      const { isSignerConnected } = useSafe()

      return (
        <>
          <h3>{isSignerConnected ? 'Signer is connected' : 'No signer connected'}</h3>
        </>
      )
    }

    export default IsSignerConnected
    ```
  </Tabs.Tab>
</Tabs>

`boolean`
