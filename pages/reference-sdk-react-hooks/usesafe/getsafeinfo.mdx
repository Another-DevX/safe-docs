### `getSafeInfo`

Returns the information associated with the Safe connected to the [`SafeProvider`](../safeprovider.mdx).

## Usage

{/* <!-- vale off --> */}

<Tabs items={['example.tsx']}>
  <Tabs.Tab>
    ```typescript
    import { useSafeInfo } from '@safe-global/safe-react-hooks'

    function SafeInfo() {
      const { getSafeInfo } = useSafe()
      const { data, error, isLoading } = getSafeInfo()

      if (isLoading) return <p>Loading...</p>
      
      if (error) return <p>Error: {error.message}</p>

      return JSON.stringify(data, null, 2)
    }

    export default SafeInfo
    ```
  </Tabs.Tab>
</Tabs>

{/* <!-- vale on --> */}

## Returns

`UseSafeInfoReturnType`

- A [query result](https://tanstack.com/query/v5/docs/framework/react/reference/useQuery) object containing the Safe account information in the `data` property.
- The `data` property [format](https://github.com/safe-global/safe-react-hooks/blob/6f620f2f9492d518af332cb7f21646beef4fb42f/src/types/index.ts#L32-L38)

## Parameters

- **Type**: `UseSafeInfoParams`

Parameters to customize the hook behavior.

```typescript
const result = getSafeInfo(params)
```

### `config` (Optional)

- **Type**: [`SafeConfig`](../safeconfig.mdx)

The configuration used instead of the one from the nearest [`SafeProvider`](../safeprovider.mdx).

<Tabs items={['index.tsx', 'config.ts']}>
