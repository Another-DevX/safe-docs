### `getBalance`

Returns the balance information of the Safe connected to the [`SafeProvider`](../safeprovider.mdx).

## Usage

{/* <!-- vale off --> */}

<Tabs items={['example.tsx']}>
  <Tabs.Tab>
    ```typescript
    import { useSafe } from '@safe-global/safe-react-hooks'

    function SafeBalance() {
      const { getBalance } = useSafe()
      const { data, error, isLoading } = getBalance()

      if (isLoading) return <>Loading...</>

      if (error) return <>Error: {error.message}</>

      return (
        <>
          {JSON.stringify(data, null, 2)}
        </>
      )
    }

    export default SafeBalance
    ```
  </Tabs.Tab>
</Tabs>

{/* <!-- vale on --> */}

## Returns

[`UseBalanceReturnType`](https://wagmi.sh/react/api/hooks/useBalance#return-type)

The [query result](https://tanstack.com/query/v5/docs/framework/react/reference/useQuery) object with the Safe balance in the `data` property.

## Parameters

- **Type**: `UseBalanceParams`

Parameters to customize the hook behavior.

```typescript
const result = getBalance(params)
```

### `config` (Optional)

- **Type**: [`SafeConfig`](../safeconfig.mdx)

The configuration used instead of the one from the nearest [`SafeProvider`](../safeprovider.mdx).

{/* <!-- vale off --> */}

<Tabs items={['index.tsx', 'config.ts']}>
